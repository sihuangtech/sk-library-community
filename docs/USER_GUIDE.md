# 用户使用指南

本文档详细介绍家庭图书管理系统的功能和使用方法。

## 系统概述

家庭图书管理系统是一个基于Web的图书管理工具，帮助您：
- 管理家庭图书收藏
- 通过ISBN自动获取图书信息
- 追踪图书借阅情况
- 统计图书数据

## 系统登录

### 首次访问

1. 打开浏览器，访问 `http://localhost:3008`
2. 系统会自动跳转到登录页面
3. 使用管理员账户登录：
   - 用户名：在配置文件中设置的管理员用户名
   - 密码：在配置文件中设置的管理员密码

### 登录界面

![登录界面示例]
- 输入用户名和密码
- 点击"登录"按钮
- 登录成功后会跳转到首页

### 忘记密码

如果忘记密码，需要：
1. 编辑项目根目录下的 `config.yaml` 文件
2. 修改 `admin.password` 字段
3. 重启应用

## 主界面导航

### 导航栏

系统顶部导航栏包含以下菜单：
- **首页**：系统概览和统计信息
- **图书管理**：图书列表和管理功能
- **添加图书**：添加新图书到系统
- **借阅管理**：管理图书借阅和归还
- **系统设置**：系统配置和管理
- **登出**：退出系统

### 页脚信息

页面底部显示：
- 版权信息
- 系统版本
- 备案信息（如已配置）

## 图书管理

### 查看图书列表

1. 点击导航栏中的"图书管理"
2. 系统显示所有图书的列表
3. 列表包含以下信息：
   - 封面图片
   - 图书标题
   - 作者
   - 出版社
   - ISBN号
   - 借阅状态

### 搜索图书

在图书列表页面：
1. 使用页面顶部的搜索框
2. 输入关键词（标题、作者、ISBN等）
3. 按回车键或点击搜索按钮
4. 系统显示匹配的图书

### 筛选图书

- 使用分类筛选器选择特定分类
- 使用借阅状态筛选（可借阅/已借出）
- 组合多个筛选条件

### 查看图书详情

1. 在图书列表中点击任意图书
2. 系统显示图书详细信息：
   - 基本信息（标题、作者、出版社等）
   - 出版信息（出版日期、页数、装帧等）
   - 价格信息
   - 图书描述
   - 借阅历史

### 编辑图书信息

1. 在图书详情页面点击"编辑"按钮
2. 修改需要更新的字段
3. 点击"保存"按钮确认更改
4. 系统显示更新成功的提示

### 删除图书

1. 在图书详情页面点击"删除"按钮
2. 系统弹出确认对话框
3. 确认删除后，图书将从系统中移除
4. **注意**：删除操作不可恢复

## 添加图书

### 通过ISBN添加（推荐）

1. 点击导航栏中的"添加图书"
2. 在ISBN搜索框中输入图书的ISBN号
3. 点击"搜索"按钮
4. 系统自动获取图书信息并填充表单
5. 检查信息是否正确，可手动修改
6. 点击"添加图书"按钮保存

### 手动添加

1. 在添加图书页面选择"手动输入"
2. 填写图书的基本信息：
   - **必填项**：ISBN、标题、作者
   - **可选项**：出版社、出版日期、价格、页数、分类、描述等
3. 上传或输入封面图片URL
4. 点击"添加图书"按钮保存

### 批量添加

目前系统不支持批量添加功能，需要逐本添加图书。

### 添加注意事项

- ISBN号必须唯一，不能重复
- 建议使用ISBN搜索功能以获得准确信息
- 封面图片URL应该是可访问的网络地址

## 借阅管理

### 借出图书

1. 在图书列表或详情页面找到要借出的图书
2. 点击"借出"按钮
3. 填写借阅信息：
   - 借阅人姓名
   - 预计归还日期
4. 点击"确认借出"
5. 图书状态更新为"已借出"

### 归还图书

1. 在借阅管理页面查看已借出的图书
2. 找到要归还的图书
3. 点击"归还"按钮
4. 确认归还操作
5. 图书状态更新为"可借阅"

### 查看借阅记录

1. 点击导航栏中的"借阅管理"
2. 查看当前借阅情况：
   - 已借出图书列表
   - 借阅人信息
   - 借阅日期
   - 预计归还日期
   - 逾期状态

### 逾期提醒

系统会自动标识逾期图书：
- 超过预计归还日期的图书会显示为红色
- 在借阅管理页面可以快速识别逾期图书

## 常见问题

### 登录问题

**Q: 忘记管理员密码怎么办？**
A: 编辑 `config.yaml` 文件中的 `admin.password` 字段，然后重启应用。

**Q: 登录后自动退出？**
A: 检查系统时间是否正确，或者增加会话有效期设置。

### 图书管理问题

**Q: ISBN搜索不到图书信息？**
A: 
1. 检查ISBN号是否正确
2. 确认网络连接正常
3. 验证API密钥是否有效

**Q: 图书封面不显示？**
A: 
1. 检查图片URL是否可访问
2. 确认图片格式是否支持
3. 尝试重新上传图片

**Q: 无法删除图书？**
A: 确认图书当前没有被借出，已借出的图书需要先归还。

### 系统性能问题

**Q: 系统运行缓慢？**
A: 
1. 检查数据库大小
2. 清理旧的备份文件
3. 重启应用

**Q: 搜索速度慢？**
A: 
1. 使用更精确的搜索条件
2. 考虑对图书进行分类管理

### 数据安全问题

**Q: 如何防止数据丢失？**
A: 
1. 定期使用系统备份功能
2. 将备份文件保存到外部存储
3. 在重要操作前手动备份

**Q: 如何恢复误删的数据？**
A: 
1. 使用最近的备份文件恢复
2. 参考 [更新文档](UPDATE.md) 中的回滚操作
